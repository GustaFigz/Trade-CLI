# MSS — Market Structure Shift

## Definição

Um **Market Structure Shift (MSS)** — também chamado **Change of Character (ChoCH)** no contexto SMC — é quando o preço quebra a estrutura de mercado na direcção oposta à tendência anterior, sinalizando uma possível mudança de direcção.

**Diferença BOS vs MSS/ChoCH:**
- **BOS (Break of Structure):** Quebra na mesma direcção da tendência (confirma continuação)
- **MSS/ChoCH:** Quebra na direcção oposta (sinal de reversão)

## Visualização

```
Uptrend → MSS (bearish):
   HH   HH   HL      ← último HL
        ↓ Preço desce abaixo do último HL = MSS BEARISH
   (agora é lower low → possível reversão)

Downtrend → MSS (bullish):
        ↑ Preço sobe acima do último LH = MSS BULLISH
   LH   LH   HL
```

## MSS vs ChoCH

| Termo | Sistema | Significado |
|-------|---------|------------|
| MSS | ICT | Market Structure Shift — após liquidity sweep |
| ChoCH | SMC | Change of Character — equivalente ao MSS |
| BOS | SMC | Break of Structure — confirma tendência |

## Uso Prático

### MSS como Confirmação de Entrada
1. [[conceitos/liquidity-sweep]] ocorre
2. **MSS confirma** que o movimento foi real (não fake)
3. Entrar em pullback para OB ou FVG
4. SL abaixo/acima do swing low/high do MSS

### Sequência Completa
```
Sweep BSL → MSS bearish → Pullback para Bearish OB → SHORT
Sweep SSL → MSS bullish → Pullback para Bullish OB → LONG
```

## Quando o MSS é FORTE

✅ MSS forte quando:
1. Ocorre após sweep de liquidez significativa
2. Vela de MSS é impulsiva (grande corpo, pouco wick)
3. Alinhado com HTF bias
4. Ocorre durante killzone

❌ MSS fraco quando:
1. Sem sweep anterior
2. Vela de MSS pequena (indecisão)
3. Contra HTF bias
