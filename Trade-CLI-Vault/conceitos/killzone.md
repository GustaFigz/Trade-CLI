# Killzone — Janelas de Alta Probabilidade ICT

## Definição

**Killzones** são janelas temporais específicas onde os **grandes players institucionais** estão activos e a liquidez é máxima. Durante estas janelas, os setups ICT/SMC têm maior probabilidade de funcionar.

**Premissa:** Fora das killzones, o mercado é mais "aleatório" e dominado por algoritmos de menor escala.

## As 4 Killzones

### 🇬🇧 London Open Killzone
| Campo | Detalhe |
|-------|---------|
| **Horário UTC** | 08:00 – 11:00 |
| **Horário Lisboa (inverno)** | 08:00 – 11:00 |
| **Horário Lisboa (verão)** | 09:00 – 12:00 |
| **Ativos principais** | [[ativos/EURUSD]], [[ativos/USDJPY]] |
| **Características** | London range estabelece, sweeps de Asian range frequentes |
| **Setup típico** | Sweep de Asian high/low → MSS → Entry |

### 🇺🇸 New York Open Killzone
| Campo | Detalhe |
|-------|---------|
| **Horário UTC** | 13:00 – 16:00 |
| **Horário Lisboa (inverno)** | 13:00 – 16:00 |
| **Horário Lisboa (verão)** | 14:00 – 17:00 |
| **Ativos principais** | Todos — [[ativos/EURUSD]], [[ativos/US30]], [[ativos/NAS100]] |
| **Características** | Maior volume do dia, reversões ou continuações do London |
| **Setup típico** | Sweep de London high/low → MSS → Entry |

### 🌙 New York Close Killzone
| Campo | Detalhe |
|-------|---------|
| **Horário UTC** | 19:00 – 20:00 |
| **Ativos principais** | [[ativos/EURUSD]], [[ativos/US30]] |
| **Características** | Position squaring, reversões do NY session |

### 🌏 Asian / Tokyo Killzone
| Campo | Detalhe |
|-------|---------|
| **Horário UTC** | 23:00 – 02:00 |
| **Ativos principais** | [[ativos/USDJPY]] |
| **Características** | Range consolidação, low liquidity, cria BSL/SSL para London |

## Como Usar Killzones

### Processo
```
1. Verificar qual killzone está activa
2. Identificar o range da sessão anterior (Asian range, London range)
3. Aguardar sweep dos extremos desse range
4. Confirmar MSS
5. Entrar no OB ou FVG
```

### Regra de Ouro
> **Nunca entrar a meio de uma sessão sem context — espera sempre pela killzone.**
